AUTONOMOUS POSTMAN ROBOT

The growing need of delivering and receiving packages is definitely noticeable and with the pandemic that is going on and with social distancing applied we had to think of a safe way to get the job done. This project aims to deal with the issue of delivering small packages over a specific distance by the help of an autonomous robot that will be responsible for storing it inside small cargo space and delivering it to the person calling for it all according to the path this robot is following

Scope

  • Select the suitable material to build the robot.
  • The potential of postman acts during delivering the packages, including:
    • Saving the packages
    • Delivering to the client location.
  • The robot is Autonomous, which means it can function by itself.
  • Design that will allow the robot to avoid obstacles and climbs stairs.

Objectives & Goals

The purpose of this project is to develop and design a Postman Robot guided by these following objectives:

  1. To determine the use of multiple sensors for various sensing on the robot
  2. To design a robot that is able to avoid obstacles and Climb Stairs.
  3. Delivering the packages without any damages
  4. The robot will have a cargo place that can contain a 3 Kg package

Constraints

Design Specification

Make a decision on the type of sensor (i.e. Kinect), interface, and any other essential devices.

Come up with the most effective way of placing the sensors and other necessary pieces around the robot.

Work out an efficient logic or intelligent sequence to avoid collision and obstacles.

Design the robot, including “materials, mechanisms”

Performance Specifications

On the performance aspect, the project must consist of a reliable design that will allow the robot to avoid obstacles and climbs stairs.
